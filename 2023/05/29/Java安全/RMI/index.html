Java RMI

Java RMI ( Remote Method Invocation ) 远程方法调用，它允许在一个 Java 虚拟机中运行的对象调用在另一个 Java 虚拟机中运行的对象上的方法。

RMI 分布式应用程序：

  • RMI Server ：创建一些远程对象
  • Registry ：服务端调用注册表以将名称与远程对象关联
  • RMI Client：客户端在服务器的注册表中按名称查找远程对象，然后对其调用方法
