这里就简单过一下修改后的模板：

  1. 输入信息
  2. 根据漏洞类型判断其是文件读取还是能执行命令
  3. 执行命令
    1. 有无回显
      1. 有回显 => 通过 self.exec 去获取每个命令的执行结果
      2. 无回显 => 通过下载远程的 bat / sh 文件去执行 将收集到的系统信息保存到静态文件目录下 然后在读取解析获取命令执行结果
    2. 写 webshell ?
      1. 通过 echo 写 => 选择语言 => 生成对应的马的内容 key password 传入到模板 => self.exec 去写
      2. 通过下载 => 选择语言 => 当前目录下生成一个对应的马的文件 => 上传获取 URL （手动）=> 模板生成后修改下载的 url （ 待优化，自动上传到未授权的 aws 并返回 url 直接填入模板 ）
